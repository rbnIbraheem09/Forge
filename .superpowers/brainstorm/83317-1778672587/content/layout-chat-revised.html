A few choices baked in — let me know if any feel off:

  • Chat history is kept, so you can iterate ("make her a redhead", "shift to rooftop"). The latest response is highlighted; earlier ones scroll above.
  • Tag library is clickable — click a tag and it drops into the message box as part of your description (e.g. "include golden_hour"). It's a reference + nudge tool, not the main interaction.
  • Temperature slider 0–2 lives in the input dock alongside the send button.
  • Each AI response shows positive and negative as two separately copyable blocks. Plus a "Copy All" and "Save preset" on the latest message.
