Database Schema Changes

New / Replaced Tables
main_gens — replaces generations
title, hero_image_path, hero_color, pinned, notes, tags
iterations — children of main_gen
main_gen_id, iteration_number, image_path, all metadata, starred, notes, tags
iteration_loras — replaces generation_loras
iteration_id, lora_id, weight
iteration_custom_fields — new
iteration_id, field_key, field_value
global_field_templates — new
field_key (pinned global custom fields)
inbox_items — new
image_path, extracted_metadata JSON, detected_at
Unchanged Tables
models — checkpoints (as-is)
loras — LoRA models (as-is)
extras — free-form notes (as-is)
settings — folder paths (as-is)
